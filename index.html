1.安装git 官方地址：https://git-scm.com/
安装好之后，鼠标右键可以看到：Git Bash Here，点击后打开了。
输入：git –version

2.安装node.js 官方地址：https://nodejs.org/en/download/
安装好之后，配置环境变量，并在终端里面输入：node -v

1.配置ssh
先生成ssh key ssh-keygen -t rsa -C “邮件地址”

输入生成命令：ssh-keygen -t rsa -C “xujinjievip@qq.com“后。连续回车三次即可：

可以看到在用户文件夹下生成了一个ssh文件夹：

进入到ssh文件夹，复制id_rsa.pub文件里全部内容：

打开github主页，点击个人设置，点击左侧的SSH and GPG keys，点击New SSH key ：

将id_rsa.pub复制的内容粘贴到key中，title随便起一个就行

输入ssh -T git@github.com测试命令后，接着输入yes，出现下面这个提示说明成功：
