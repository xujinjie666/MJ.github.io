利用Github+hexo搭建自己的个人博客

1.环境准备

在搭建Github博客之前，先准备一下环境（按照下方的顺序来安装，别弄混了）。
1.安装git 官方地址：https://git-scm.com/
安装好之后，鼠标右键可以看到：Git Bash Here，点击后打开了。
输入：git –version

2.安装node.js 官方地址：https://nodejs.org/en/download/
安装好之后，配置环境变量，并在终端里面输入：node -v

3.安装Hexo 安装命令：

测试是否安装成功，命令：hexo -v

4.安装hexo依赖 命令：

2.git配置SSH key

1.配置ssh
先生成ssh key ssh-keygen -t rsa -C “邮件地址”
备注：这里的邮件地址是github账号绑定的邮件地址

输入生成命令：ssh-keygen -t rsa -C “xujinjievip@qq.com“后。连续回车三次即可：

可以看到在用户文件夹下生成了一个ssh文件夹：

进入到ssh文件夹，复制id_rsa.pub文件里全部内容：

打开github主页，点击个人设置，点击左侧的SSH and GPG keys，点击New SSH key ：

将id_rsa.pub复制的内容粘贴到key中，title随便起一个就行

输入ssh -T git@github.com测试命令后，接着输入yes，出现下面这个提示说明成功：

配置账号密码：
$ git config –global user.name “xujinjie666” # 你的github用户名
$ git config –global user.email “xujinjievip@163.com“ # 填写你的github注册邮箱

博客搭建成功 在本地的启动：

3.部署到github

新建一个github仓库：
给仓库起个名称，注意！注意！注意！

仓库名称格式：用户名.github.io

我的的仓库名称为：MJ.github.io

因为这个仓库名称将作为你github博客的访问地址：

编辑_config.yml，_config.yml在博客存放目录下：

repository仓库地址改为自己的。
branch看自己的github仓库是master还是main 这里是main，所以就填写main

发布到github 命令：hexo d

在浏览器访问 https://xujinjie666.github.io/
